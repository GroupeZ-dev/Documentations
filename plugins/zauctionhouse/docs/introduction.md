# zAuctionHouse

Welcome to the zAuctionHouse documentation!

zAuctionHouse is a powerful auction house plugin for Minecraft servers that integrates seamlessly with zMenu for customizable inventory interfaces.

## Features

- **Auction System** - List items for sale with customizable prices and durations
- **Bidding System** - Allow players to bid on items
- **Categories** - Organize items into categories for easy browsing
- **Search** - Search for specific items
- **History** - Track past purchases and sales
- **zMenu Integration** - Fully customizable interfaces using zMenu
- **Economy Support** - Works with Vault and other economy plugins

## Requirements

- Java 17+
- Minecraft 1.20.5+
- zMenu plugin
- Vault (for economy)
