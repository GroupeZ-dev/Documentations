# zTransaction

Welcome to the zTransaction documentation!

zTransaction is a transaction tracking plugin for Minecraft servers that helps you monitor and manage player economic activities.

## Features

- **Transaction Logging** - Track all player transactions
- **Statistics** - View detailed statistics per player
- **History** - Browse transaction history
- **PlaceholderAPI** - Display transaction data with placeholders
- **Webhook Support** - Send notifications to Discord
- **Database Support** - MySQL and SQLite support

## Requirements

- Java 17+
- Minecraft 1.20.5+
- Vault (optional, for economy integration)
