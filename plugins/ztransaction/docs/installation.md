# Installation

## Requirements

Before installing zTransaction, make sure you have:

- Java 17 or higher
- Minecraft server 1.20.5+

## Installation Steps

1. Download zTransaction from [SpigotMC](https://www.spigotmc.org/members/maxlego08.45892/)
2. Place the JAR file in your server's `plugins` folder
3. Restart your server
4. Configure the plugin in `plugins/zTransaction/config.yml`

## First Start

On first start, zTransaction will:
- Create default configuration files
- Set up the database

## Verification

To verify the installation:
```
/ztransaction help
```
