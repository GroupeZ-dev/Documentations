# zMenu+

**zMenu+** is the premium addon for zMenu that adds powerful advanced features to enhance your inventory creation experience.

## What is zMenu+?

zMenu+ extends the capabilities of the free zMenu plugin with additional features designed for server owners who want even more control and functionality in their menus.

## Features

zMenu+ includes exclusive features that are not available in the free version:

### Advanced Button Types
Additional button types for more complex menu behaviors.

### Enhanced Actions
Extra action types for more interactive menus.

### Premium Support
Priority support for zMenu+ customers.

### Early Access
Get access to new features before they're released in the free version.

## Purchase

You can purchase zMenu+ from:

**Minecraft Inventory Builder**: [https://minecraft-inventory-builder.com/resources/zmenu-addon](https://minecraft-inventory-builder.com/resources/zmenu-addon)

## Installation

1. Purchase and download zMenu+ from the link above
2. Make sure zMenu is already installed on your server
3. Place the zMenu+ JAR file in your `plugins/` folder
4. Restart your server

:::info Requires zMenu
zMenu+ is an addon and requires the base zMenu plugin to function. Make sure you have zMenu installed first.
:::

## Frequently Asked Questions

### Is zMenu+ required to use zMenu?

No, zMenu is fully functional as a free plugin. zMenu+ is optional and adds extra features.

### Can I try zMenu+ before buying?

Join the Discord server to see examples and ask questions about what zMenu+ offers.

### Is there a refund policy?

Please check the Minecraft Inventory Builder website for their refund policy.

### Will my inventories work with zMenu+?

Yes! zMenu+ is fully compatible with your existing zMenu configurations. It only adds new features - it doesn't change how existing features work.
