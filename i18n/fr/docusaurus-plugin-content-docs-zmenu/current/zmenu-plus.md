# zMenu+

**zMenu+** est l'addon premium pour zMenu qui ajoute de puissantes fonctionnalites avancees pour ameliorer votre experience de creation d'inventaires.

## Qu'est-ce que zMenu+ ?

zMenu+ etend les capacites du plugin gratuit zMenu avec des fonctionnalites supplementaires concues pour les proprietaires de serveurs qui veulent encore plus de controle et de fonctionnalites dans leurs menus.

## Fonctionnalites

zMenu+ inclut des fonctionnalites exclusives qui ne sont pas disponibles dans la version gratuite :

### Types de boutons avances
Types de boutons supplementaires pour des comportements de menu plus complexes.

### Actions ameliorees
Types d'actions supplementaires pour des menus plus interactifs.

### Support premium
Support prioritaire pour les clients zMenu+.

### Acces anticipe
Obtenez l'acces aux nouvelles fonctionnalites avant leur sortie dans la version gratuite.

## Achat

Vous pouvez acheter zMenu+ depuis :

**Minecraft Inventory Builder** : [https://minecraft-inventory-builder.com/resources/zmenu-addon](https://minecraft-inventory-builder.com/resources/zmenu-addon)

## Installation

1. Achetez et telechargez zMenu+ depuis le lien ci-dessus
2. Assurez-vous que zMenu est deja installe sur votre serveur
3. Placez le fichier JAR de zMenu+ dans votre dossier `plugins/`
4. Redemarrez votre serveur

:::info Necessite zMenu
zMenu+ est un addon et necessite le plugin de base zMenu pour fonctionner. Assurez-vous d'avoir installe zMenu en premier.
:::

## Questions frequentes

### zMenu+ est-il requis pour utiliser zMenu ?

Non, zMenu est entierement fonctionnel en tant que plugin gratuit. zMenu+ est optionnel et ajoute des fonctionnalites supplementaires.

### Puis-je essayer zMenu+ avant d'acheter ?

Rejoignez le serveur Discord pour voir des exemples et poser des questions sur ce que zMenu+ offre.

### Y a-t-il une politique de remboursement ?

Veuillez consulter le site Minecraft Inventory Builder pour leur politique de remboursement.

### Mes inventaires fonctionneront-ils avec zMenu+ ?

Oui ! zMenu+ est entierement compatible avec vos configurations zMenu existantes. Il ajoute seulement de nouvelles fonctionnalites - il ne change pas le fonctionnement des fonctionnalites existantes.
